SNMP Query Tool

Author: Dalton Ponder
Current Version: 1.01

=====================

How to Use:

Since this is a Java application in a .jar archive, you can run this application on any platform that has the Java Runtime Environment 8 (JRE 1.8) installed on it.

	- To start this application, on any platform you will need to open the platform's command line/terminal
	- You will then need to navigate to the directory containing the SNMPQueryTool.jar file
	- Once in the correct directory, you will need to run Java's JRE .jar run script (seen below without the quotation marks):
		"java -jar SNMPQueryTool.jar"
